The Speech Recognition Virtual Kitchen

Welcome to the Speech Recognition Virtual Kitchen!

The Speech Recognition Virtual Kitchen is dedicated to improving community research and education infrastructure in speech recognition. The >kitchen< environment hosts Virtual Machines (VMs) that provide a consistent environment for experimentation. We liken the VMs to a >kitchen< because they provide the infrastructure into which one can install >appliances< (e.g., speech recognition toolkits), “recipes” (scripts for creating state-of-the art systems), and “ingredients” (language data). The kitchen provides fully configured VMs with documented baselines, so that you can start experimenting right away, without having to download, configure, compile, verify, update, recompile, etc ... >stuff<.

The Speech Kitchen can be used in several ways:

  • Download and run a Virtual Machine directly on your computer using Virtualbox or VMWare. This is like a computer within your computer.
  • Run one of our open-sourced images on Amazon EC2.
  • Check out one of our repositories on Github, and build the VM yourself using Vagrant. A smaller download and you can easily change the configuration of your VM and re-create it.

If you have something that you would like to contribute and share with others - please drop us a line! We'll be happy to re-distribute (in fact, we do not have resources to develop a lot of stuff ourselves).
